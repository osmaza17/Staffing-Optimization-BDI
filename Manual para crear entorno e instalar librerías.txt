================================================================
 MANUAL DE INSTALACION Y EJECUCION (PYTHON / VS CODE)
 PROYECTO: Staffing-Optimization-BDI
================================================================

REQUISITO PREVIO:
Abre la carpeta del proyecto en VS Code y abre una terminal (Ctrl + ñ).


----------------------------------------------------------------
PASO 1: LIMPIEZA INICIAL
----------------------------------------------------------------
Si en tu carpeta ya existe una carpeta llamada "venv", "env" o ".venv"
(copiada de otra ubicacion), BORRALA MANUALMENTE.
Los entornos virtuales no funcionan si se mueven de sitio.


----------------------------------------------------------------
PASO 2: CREAR EL ENTORNO VIRTUAL
----------------------------------------------------------------
Ejecuta este comando en la terminal para crear el espacio aislado:

    python -m venv venv


----------------------------------------------------------------
PASO 3: ACTIVAR EL ENTORNO (EN TERMINAL)
----------------------------------------------------------------
Ejecuta esto para activar el modo aislado en la consola actual:

    .\venv\Scripts\Activate

(Si sale error rojo de scripts, ejecuta:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
y repite el comando de activar).


----------------------------------------------------------------
PASO 4: VINCULAR EL ENTORNO A VS CODE (¡IMPORTANTE!)
----------------------------------------------------------------
Para que el boton de "Play" y el autocompletado funcionen con las
librerias que vamos a instalar, debes decirselo a VS Code:

1. Presiona las teclas: Ctrl + Shift + P
2. Escribe y selecciona: Python: Select Interpreter
3. En la lista, busca y selecciona la opcion que dice "venv"
   (Suele aparecer con una estrella o decir "Recommended").

NOTA: Si VS Code te pregunta si quieres seleccionarlo para el
workspace, di que SI.


----------------------------------------------------------------
PASO 5: INSTALAR LIBRERIAS
----------------------------------------------------------------
Ahora que VS Code y la terminal miran al sitio correcto, instala:

>>> OPCION A (Si tienes requirements.txt):
    pip install -r requirements.txt

>>> OPCION B (Manual):
    pip install flet pulp openpyxl highspy


----------------------------------------------------------------
PASO 6: GUARDAR CONFIGURACION (Solo si usaste Opcion B)
----------------------------------------------------------------
    pip freeze > requirements.txt


----------------------------------------------------------------
PASO 7: EJECUTAR EL PROGRAMA
----------------------------------------------------------------
Ya puedes darle al boton de "Play" de VS Code o escribir:

    python main.py
